Run: `npm install googleapis google-auth-library --legacy-peer-deps`

Create `lib/googleSheets.ts` with the following:

1. A `CheckInPayload` interface:
   ```typescript
   export interface CheckInPayload {
     nome: string;
     cognome: string;
     email: string;
     scuola: string;
     dataOra: string;       // Italian formatted datetime string from client
     decodedText: string;   // raw QR content — tipo resolved server-side
   }
   ```

2. Private constant `HEADERS = ['Nome', 'Cognome', 'Email', 'Scuola', 'Data e Ora', 'Tipo']`

3. Private `getSheetsClient()` function:
   - Reads `process.env.GOOGLE_SERVICE_ACCOUNT_KEY` — throws if missing
   - Parses with `JSON.parse(raw)` — no `.replace()` calls on private_key
   - Creates `new google.auth.JWT({ email: credentials.client_email, key: credentials.private_key, scopes: ['https://www.googleapis.com/auth/spreadsheets'] })`
   - Returns `google.sheets({ version: 'v4', auth })`

4. Private `ensureHeaders(sheets, spreadsheetId)` function:
   - Calls `sheets.spreadsheets.values.get({ spreadsheetId, range: 'A1:F1' })`
   - If `res.data.values` is falsy or empty array, calls `sheets.spreadsheets.values.update` with `range: 'A1:F1'`, `valueInputOption: 'RAW'`, `requestBody: { values: [HEADERS] }`

5. Exported `appendCheckin(payload: CheckInPayload): Promise<void>` function:
   - Reads `process.env.GOOGLE_SPREADSHEET_ID` — throws if missing
   - Calls `getSheetsClient()`
   - Resolves `tipo`: compare `payload.decodedText` against `process.env.ENTRANCE_QR_VALUE` and `process.env.EXIT_QR_VALUE`. If matches entrance → `'Entrata'`, if matches exit → `'Uscita'`, otherwise → `'Entrata'` as safe default (log to console with `[checkin] unknown QR value`)
   - Calls `ensureHeaders(sheets, spreadsheetId)`
   - Calls `sheets.spreadsheets.values.append` with:
     - `range: 'A:F'`
     - `valueInputOption: 'USER_ENTERED'`
     - `insertDataOption: 'INSERT_ROWS'`
     - `requestBody.values: [[payload.nome, payload.cognome, payload.email, payload.scuola, payload.dataOra, tipo]]`
   - No retry in this function — retry lives in route handler

6. Use proper TypeScript types from googleapis. Import: `import { google } from 'googleapis'`

Note: `google-auth-library` is a peer dep — import via googleapis is sufficient; no direct import from google-auth-library needed.

**Part A — Create `app/api/checkin/route.ts`:**

Create `app/api/checkin/` directory and `route.ts` with:

```typescript
import { NextRequest } from 'next/server';
import { appendCheckin, CheckInPayload } from '@/lib/googleSheets';

export async function POST(request: NextRequest) {
  try {
    const body: CheckInPayload = await request.json();

    // Validate required fields
    if (!body.nome || !body.cognome || !body.email || !body.scuola || !body.dataOra || !body.decodedText) {
      return Response.json({ error: 'Dati mancanti.' }, { status: 400 });
    }

    // 1 immediate retry on transient failure
    try {
      await appendCheckin(body);
    } catch (firstErr) {
      console.error('[checkin] First attempt failed, retrying:', firstErr);
      await appendCheckin(body); // throws on second failure — caught below
    }

    return Response.json({ ok: true });
  } catch (err) {
    console.error('[checkin] Sheets write failed:', err);
    return Response.json(
      { error: 'Errore di sincronizzazione con Google Sheets.' },
      { status: 500 }
    );
  }
}
```

Runtime must be Node.js (NOT Edge) because googleapis uses Node.js crypto APIs. Add at top of file before the import block:
```typescript
export const runtime = 'nodejs';
```
